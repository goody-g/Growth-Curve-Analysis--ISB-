gSliceR Documentation

View Slices of Parameter Space

Description

Create a set of scatter plots to visualize distribution of parameters.

Usage

gSlice(g.analysis, compare=NULL, name="slice", control=gControl())

Arguments

g.analysis An object of class "gAnalysis" containing the analysis of a set of wells to analyze.
compare A list of two character vectors, one compare$info and one compare$param of attributes to list on the gSlice graph. If null, control$compare is used.
svg A logical object stating whether to output to R or an svg.
name The name of the saved slice file if an SVG output is made.
control An object of class "gControl" providing options for gSlice. The colors for the gSlice graph are specified by control$color.by.

Details

This function provides a visualization of parameters for the validation of parameter estimation. Clumpings by color indicate more sucessful parameters - if estimated parameters fit graphs from similar environmental conditions, they are more likely to be accessing something meaningful in the biology. This visualization also alows the developer to notice outliers in the parameters and investigate them using the svg plot.

Author(s)

Goodwin Gibbins

References

Draws on the Grofit Package, http://www.jstatsoft.org/v33/i07/paper

See Also

GCAF, gAnalysis, plot.gAnalysis

Examples

#Prepare g.analysis
matches <- list(Background="NRC-1", Date=20080415, Media="CM")
chosen.wells <- gChoose(matches)
g.analysis <- gAnalysis(chosen.wells)

#Visualze the Slice
gSlice(g.analysis)
