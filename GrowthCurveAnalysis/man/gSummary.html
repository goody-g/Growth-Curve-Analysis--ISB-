gSummaryR Documentation

Summarize list("gAnalysis") Results

Description

Provides a rapid summary of the metainformation in a set of wells analyzed by gAnalysis to provide a check for gChoose etc.

Usage

gSummary(g.analysis)

Arguments

g.analysis An object of class "gAnalysis" which contains metainformation and derived parameters for a set of growth curves chosen by gChoose.
control An object of class "gControl" providing options for gSummary, specifically control$no.display to supress metainformation categories.

Details

This function should be used as a simple user-check for for the contents of an object returned from gAnalysis, for example to check whether an environmental perterbation was included inadvertantly.

Value

to.display A list of metainformation categories and attributes with their number of occurences. NAs are supressed.
