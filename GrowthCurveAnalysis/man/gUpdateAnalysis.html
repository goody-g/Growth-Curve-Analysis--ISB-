gUpdateAnalysisR Documentation

Updates Saved gAnalysis Objects

Description

Allows for a database-wide update of saved gAnalysis objects which should be used for quicker analysis of sets of curves.

Usage

gUpdateAnalysis(replace.old=FALSE, replace.id="Test", control=gControl())

Arguments

replace.old Logical indicating whether to save old gAnalysis file in an Archive folder or write over.
replace.id A name to append to Analysis for archived gAnalysis class objects, preferably date in YYYYMMDD form.
control An object of class "gControl" providing options for gSummary

Details

This function has not been integrated into the GCAF pathway yet - ultimately, the saved gAnalysis objects would be searched and used for analysis such as gSlice rather than performing gAnalysis again.
