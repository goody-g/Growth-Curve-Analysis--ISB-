low.integrateR Documentation

Function to estimate the area under a curve given as respecting x and y(x) values

Description

Approximates the integral of function by calculating the area under a smoothed spline fitted to the given x and y(x) values.

Usage

low.integrate(x, y)

Arguments

x Numeric vector.
y Numeric vector, corresponding to x.

Details

The function uses the the R internal function smooth.spline.

NOTE: This function was originally part of the Grofit package and has been copied for use in new package. This documentation is also from the Grofit package. In GCAF it is used in gSplineFit.

Value

Area under the smoothed spline.
