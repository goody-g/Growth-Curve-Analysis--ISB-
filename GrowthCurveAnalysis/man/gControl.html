gControlR Documentation

Define available GCAF options

Description

Returns an object of class gControl with options for GCAF functions, either providing defaults or user-defined options.

Usage

gControl(color.by="Date", spar=.4)

Arguments

root.folder The location of the database folder. Updated by user upon initial download of GCAF functions.
export.folder The location of a folder for exported gAnalysis spreadsheets from gExport.
spar The smoothing parameter to use with smooth.spline in gSplineFit
cv.spline Logical, indicates whether to use ordinary (TRUE) or the faster 'generalized' cross-validation (FALSE) in smooth.spline().
cv.bs Logical, indicates whether to use ordinary (TRUE) or the faster 'generalized' cross-validation (FALSE) in smooth.spline() for bootstrapping.
num.bs.samp Number of bootstrapping samples to run as part of analysis.
color.by The meta-information to color a well according to, for example, Background, Date etc. in gSlice.
num.bs.samp Number of bootstrapping samples to run as part of analysis.
export.folder The location of a folder for exported gAnalysis spreadsheets from gExport.
no.display Information parameters to supress in gSummary.
export.folder Information and parameters to supress on plots.

Details

This function can be either edited to easily change defaults or passed options for changing options in single runs. Any new options which are written into the package should be added to the this function to simplify maitenence.

The function also makes a list of all the experiments based on the appropriately-named folders in the database folder

When adding new control options, be sure to add to list of outputs!

Value

g.control An object containing all the options listed in the arguments and also a list of date and initials for every experiment in the database as defined by .

Author(s)

Goodwin Gibbins

References

Draws on the Grofit Package, http://www.jstatsoft.org/v33/i07/paper

See Also

GCAF, gAnalysis

Examples

# Contains default settings
def.control <- gControl()

#Or can be updated by user for a specific run.
my.control <- gControl(num.bs.samp=30, color.by="Media", spar=.3)
