gInfoR Documentation

Gather Metainformation for a Curve

Description

As part of the gAnalysis function, adds metainformation from the database file associated with a chosen curve.

Usage

gInfo(date=20080422, initials="FYL", well.number=156)

Arguments

date The date of the experiment for the curve to find information for. One part of the three part specification of a curve.
initials The intials of the experimenter - another part of the three part specification of a curve.
well.number The well number of a curve - the final part of the three part specification of a curve.
control An object of class "gControl" providing options for gInfo.

Details

This function acts exclusively as part of the gAnalysis pathway, gathering metainformation to include with the parameters in the analysis.

Value

g.info A list of class gInfo with metainformation categories and attributes
