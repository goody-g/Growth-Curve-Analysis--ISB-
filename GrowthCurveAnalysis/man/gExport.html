gExportR Documentation

Export gAnalysis Objects

Description

Exports analyzed growth curve data for further analysis outside of GCAF.

Usage

gExport(g.analysis, name="CMura3")

Arguments

g.analysis An object or list of objects of class gAnalysis to export.
name A character vector consisting of a name for the exported analysis. Default is "LastExport".
control An object of class "gControl" providing options for gExport, specifically the designated export folder.

Details

This function provides a connection between the GCAF functions and other external analysis techniques. Exporting in a versitile the derived parameters for chosen wells is an important ability of the GCAF functions.

Value

LastExport.txt An exported spreadsheet containing metainformation and parameters for each curve included in the g.analysis input.

Author(s)

Goodwin Gibbins
