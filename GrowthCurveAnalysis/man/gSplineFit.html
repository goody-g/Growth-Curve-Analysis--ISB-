gSplineFitR Documentation

Estimate Parameters from a Spline Fitted Curve

Description

A collection of estimating functions which provide parameters to gAnalysis.

Usage

gSplineFit(time, data, control=gControl(), for.bs=F)

Arguments

time The raw time data supplied by the results file.
data The raw data supplied by the results file.
control An object of class "gControl" providing options for gSplineFit.
for.bs Logical, indicating whether the parameters are being estimated for bootstrapping or regular estimation, and so what specifications should be used for smooth.spline.

Details

This function, along with gInfo, provide the basis for the analysis carried out under gAnalysis.

To add new parameters, functions should be written which take time, date etc data and return parameters. These functions can then be easily added to gSplineFit following the directsion in the documentation and source code.

Value

spline.fit An object containing estimated parameters and original data to be incorporated into g.analysis.
