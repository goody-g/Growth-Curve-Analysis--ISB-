plot.gAnalysisR Documentation

Plot gAnalysis Objects

Description

Plots growth curves and analysis in a variety of formats such as single well views with parameters and metainformation, all wells, and svgs.

Usage

plot(g.analysis, 1)

Arguments

g.analysis.all An object of class gAnalysis with the analysis of a set of growth curves to be visualized.
well The well to display in a single well view (based on the number in the list of gAnalysis) or "all" if want a graph of all curves.
name A name to save the svg of for an SVG plot.
svg Logical, indicating whether to output to R or save an SVG plot.
graph.log Logical, indicating whether to graph on a log axis, where T is for a log axis.
control An object of class "gControl" providing options for plot.gAnalysis.

Details

This function provides several plot styles for objects of class gAnalysis, most helpfully quick visual summary of a single curve and all serves in a set in an R window.

The plots need to be developed along with the the functions. Currently, graph.log doesn't work with SVG, SVG only works with "all" plots.
